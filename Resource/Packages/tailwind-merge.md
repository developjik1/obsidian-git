# tailwind-merge 란?
- Utility function to efficiently merge [Tailwind CSS](https://tailwindcss.com/) classes in JS without style conflicts.

# tailwind-merge install
```shell
npm i tailwind-merge
```

```shell
yarn add tailwind-merge
```

```shell
pnpm i tailwind-merge
```
